Linux System Health Checker (Bash)

This project is a Bash-based Linux system health monitoring tool.
It helps system administrators and IT support engineers quickly
check the current health of a Linux system.

Features
- Displays system uptime
- Shows memory usage
- Shows disk usage
- Displays CPU load
- Lists logged-in users

Usage
1. chmod +x health_check.sh
2. ./health_check.sh

This project was built to strengthen Linux fundamentals,
system monitoring, and scripting skills.
